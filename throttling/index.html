THROTTLING

Throttling is another technique used in web development to manage the frequency of function calls. While debouncing ensures that a function is called only after a certain period of inactivity, throttling limits the rate at which a function can be called.

In throttling, even if the function is called multiple times within the specified interval, it will only execute once. Subsequent calls made during that interval are ignored until the interval has elapsed.
